Mongolian_Universal_White

@import url(https://victorlee.github.io/Mongolian_Universal_White/stylesheet.css);

몽골문자는 오직 한 폰트만 지정될수 있도록 class를 따로 만들어서 폰트를 설정해야 한다. 외부에서 import하는 stylesheet 링크는 <style>의 최상단에 배치한다.
For Mongolian script, the font must be set by creating a separate class so that only one font is specified. Stylesheet links imported from external webpages must be placed at the top of <style>.

상당히 많은 몽골문자와 만주문자가 모양이 같으나 유니코드가 다르다. 이는 두 문자체계가 글을 씀에 있어 서법이 확연히 다르기 때문이다. 몽골문자와 만주문자는 생김새만 같은 서로 상이한 체계로 보아도 무방하다. 따라서 각 유니코드 문자별로 다른 폰트가 적용되어야 하기에 두 문자는 각각 다른 폰트를 설정해 주어야 하며, 통일을 해야 할 경우 Mongolian Universal White와 같은 두 체계를 동시에 지원하는 폰트를 사용함이 좋다.
Due to different ways of writing, many of Mongolian and Manchu characters do not share the same unicode despite of their simmilar looks, thus different unicode characters are designated different font shapes. Most Mongolian/Manchu fonts support only characters of one language, therefore either the different languages should use different fonts respectively, or unify their fonts by using those that support both languages, such as Mongolian Universal White.

다만 Mongolian Universal White마저도 만주문자를 온전히 렌더링하지 않기 때문에 victorlee.github.io/manchu-fonts/를 참고함이 좋다.
Mongolian Universal White does not perfectly render certain Manchu syllabary such as ᡩᡝ
ᡨᡝ(de, te) instead of ᡩᡝ
ᡨᡝ, therefore referring to victorlee.github.io/manchu-fonts/ is recommended.

CSS example:

.m {
    font-family: 'Mongolian Universal White';
    writing-mode: vertical-lr;
    font-size: 1.1rem;
    line-height: 1.4;
    max-width: 100%;
    max-height: 11rem;
    overflow-x: auto;
    padding-top: 0rem;
    padding-bottom: 0.5rem;
    vertical-align: text-top;
    word-wrap: break-word;
}

For full-sized scrollable vertical-lr webpage:

<style>
html {
    font-size: 1.8rem;
    writing-mode: vertical-lr;
    line-height: 1.4;
    width: 100%;
    height: 100%;
    overflow-x: auto;
}
</style>
<script>
(function() {
    function scrollHorizontally(e) {
        e = window.event || e;
        var delta = Math.max(-1, Math.min(1, (e.wheelDelta || -e.detail)));
        document.documentElement.scrollLeft -= (delta*50);
        document.body.scrollLeft -= (delta*50);
        e.preventDefault();
    }
    if (window.addEventListener) {
        window.addEventListener("mousewheel", scrollHorizontally, false); // IE9, Chrome, Safari, Opera
        window.addEventListener("DOMMouseScroll", scrollHorizontally, false); // Firefox
    } else {
        window.attachEvent("onmousewheel", scrollHorizontally); // IE 6/7/8
    }
})();
</script>
